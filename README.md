# Low-Light Image Enhancement using Machine Learning

## Overview

This project focuses on enhancing low-light images to improve their visibility, brightness, and overall visual quality. The model processes poorly illuminated images and generates enhanced outputs while preserving important image details.

## Features

* Enhances low-light images automatically
* Improves image brightness and contrast
* Preserves image details and textures
* Supports image preprocessing and enhancement pipeline
* Performance evaluation using image quality metrics

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Scikit-learn
* Machine Learning
* Jupyter Notebook

## Installation

1. Clone the repository:

```bash
git clone https://github.com/ishatiwari1/Low-light-image-enhancement-.git
```

2. Navigate to the project directory:

```bash
cd Low-light-image-enhancement-
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the enhancement script:

```bash
python main.py
```

The enhanced images will be saved in the output directory.

## Results

The model significantly improves image visibility in low-light conditions by enhancing brightness, contrast, and overall image quality while maintaining important visual details.
